# silentland

Простой приватный медиахаб для локальной разработки на SQLite.

## Локальная разработка (SQLite)
1. Установи зависимости:
```bash
npm install
```
2. Запусти локальный сервер:
```bash
npm run dev:local
```
3. Проверь `APP_CONFIG` в `index.html`:
```js
window.APP_CONFIG = {
  localApiBase: "http://127.0.0.1:3000"
};
```
4. Открой сайт через Live Server с `?dev=1`.

## Dev режим
- `?dev=1` показывает сайт за заглушкой
- `?dev=0` возвращает заглушку

## Важно
Этот режим только для локальной разработки. Для доступа друзьям нужен сервер с внешним доступом.
